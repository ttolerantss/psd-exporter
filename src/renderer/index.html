Layer Colors

Color-code your PSD layer groups and layers in Photoshop to control how LiveryLab Export treats them.

Red Locked
Always present in every export. Cannot be toggled off.
Blue Department
Group whose children become options — pick one per variant. Use for departments that share the same base livery.
Orange Info
Requires manual editing per variant, such as unit numbers or callsigns.
Fuchsia Toggle
On/off checkbox per variant. Use for livery additions like K-9, HEAT, etc.
None Regular
Always visible. Not shown in the settings panel.
About

LiveryLab Export is a PSD layer variant exporter for livery designers. Load a color-coded PSD file, configure toggle and variant layers per variant, then batch export all variants as PNGs.

Part of the LiveryLab suite of tools for the flight simulation livery painting community.
